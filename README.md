## 🛠️ Tecnologias Utilizadas

* **Banco de Dados:** PostgreSQL (Hospedado via Neon Tech)
* **ORM:** Flask-SQLAlchemy
* **Front-end:** HTML5 & CSS3

## 📊 Estrutura do Banco de Dados
O sistema gerencia duas entidades principais com relacionamento de 1:N:
1. **CATEGORIA:** Classifica os tipos de ativos de T.I. (Ex: Hardware, Software, Periféricos).
2. **ITEM:** Detalha os bens patrimoniais individuais, contendo valor estimado, responsável e localização física.
